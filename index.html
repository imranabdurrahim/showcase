Dnilam

Core Features:

  • Auction products
  • Auction timing
  • Ecommerce product
  • Inventory Management
  • User control panel
  • User payment
  • User balance recharge
  • Admin control panel
  • Sub Admin facility


Key Challenges: The main challenging part was to set an auction timer. There was a feature requested for that application, that when a customer bid on a auction, it will increase that auction time for 10 seconds.

Ulger Truck

Location Based Truck Booking


Technologies: Styled Components, jQuery, Javascript, MAP API, PHP Framework(Codeigniter), MySQL, Apache, Linux


Details: The client has a truck renting business. They were looking to develop a software which are able to take rent request from email and match the location, if there is any truck avilable in that location to rent.


Core Features:

  • Grab rent request from email
  • Upload current trucks location
  • See request and truck location on a map
  • Admin control panel


Key Challenges: To grab rent request from email and match those with trucks location


Solution: The way i solved this, first i get data from email through imap. Then parse data to get targeted information like time, quantity, destination, load etc. Save those to database. Then i match with uploaded truck locations with rent request data and show them on a map.
